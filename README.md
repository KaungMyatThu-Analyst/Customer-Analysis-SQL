# E-commerce Customer Behavior Analysis 📊
## 📌 Project Overview
This project analyzes a real-world e-commerce dataset using **SQL (MYSQL)** 
to uncover insights about customer spending habits, membership types, and satisfaction levels.

## 🛠️ Tools Used 
-- ** Database: ** MYSQL
-- ** Tool: ** MYSQL Workbench
-- ** Analysis: ** SQL Queries (Joins, Group By, Case Statements, Data Cleaning)

## 🔎 Key Findings (Insights)
-- ** Top Revenue Sources:** Gold members contributte the highest revenue ($153,404), 
nearly triple the revenue of Bronze members.
-- ** Customer Satisfaction: ** 100% of Gold members reported high 
satisfaction, while 'Unsatisfied' ratings were only found in Bronze and Silver tires.
-- ** Demographics: ** The customer base is evenly distributed across major cities like New York, Los Angeles,
and Chicago.
-- ** Spending Behavior: ** 'Younger' age groups (under 30) have a higher average spend per person compared to 'Adults'.
